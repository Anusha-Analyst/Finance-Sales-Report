# Finance Sales Dashboard
This project analyzes sales performance using Power BI.

## Tools Used
- Power BI  
- SQL  
- Excel  

## Dashboard Explanation with Screenshots

### Main Page
![Main](Main%20Page.png)

The main page provides a high-level overview of the business performance. It displays key KPIs such as Total Sales, Total Profit, Total Units, and Profit Percentage. Users can interact with filters to analyze data based on different conditions like product and date.

### Detail Page
![Detail](Detail%20Page.png)

This page gives detailed insights into each product. It shows total sales, number of units sold, cost, and profit. It helps users understand which products are performing well and which are not.

### Drill Through Page
![Drill](Drill%20through%20Page.png)

The drill-through feature allows users to click on a specific product and navigate to a deeper level of analysis. This helps in identifying the exact reason behind changes in sales or profit.

### Data Modelling
![Model](Show%20Data%20Modelling.png)

The data model defines relationships between different tables used in the dashboard. A well-structured model improves performance and ensures accurate calculations.

## Key Features

- Interactive dashboard with filters and slicers  
- Drill-through functionality for detailed analysis  
- KPI cards for quick insights  
- Clean and user-friendly design
